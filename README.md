# 批量图片处理 (feishu-image-processor) v2.0

飞书多维表格字段捷径 — 对图片URL进行压缩、缩放、裁剪、格式转换、添加水印等操作。

## 功能

- **压缩**：调整图片质量(1-100)
- **缩放**：按宽/高调整尺寸
- **格式转换**：JPG/PNG/WebP/AVIF互转
- **裁剪**：指定区域裁剪
- **水印**：文字水印 + 图片水印叠加，可调位置/透明度/大小
- **智能处理**：用自然语言描述需求，AI自动处理

## 配置

1. 在多维表格中新建字段 → 探索字段捷径 → 选择「批量图片处理」
2. 配置 DeepSeek API Key
3. 选择图片URL列和处理操作
4. 运行即可批量处理

## 技术说明

使用 images.weserv.nl CDN 进行图片处理，智能模式通过 DeepSeek API 理解用户需求。
